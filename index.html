Azzyzt JEE Tools

Azzyzt JEE Tools is a collection of software tools designed to assist software developers creating software using Java Enterprise Edition 6. It is designed to be integrated into popular Java IDEs.

If you just want to use Azzyzt JEE Tools (as opposed to modify and build them), the recommended way to install the software is via an Eclipse update site.
